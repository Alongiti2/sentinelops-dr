📥 Ingestion

Normalizes AWS CloudTrail, Kubernetes audit logs, and endpoint/EDR events into one common OCSF-style schema.

🎯 Detection Engine

Sigma-style YAML rules tagged by MITRE ATT&CK technique, evaluated by a lightweight rule-matching engine.

📊 Lifecycle Dashboard

Scores every rule against replay fixtures for precision, recall, and coverage — untested rules are flagged, not hidden.

🤖 Response Automation

Triage → enrichment → containment with a human-in-the-loop gate and full audit logging.

🧠 AI Copilot

LLM-assisted investigation summaries — drafts recommendations only, never auto-executes containment.

📋 Threat Models

Written reasoning for a new K8s cluster rollout and datacenter/firmware/BMC risk — the thinking, not just the code.
